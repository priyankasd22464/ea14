Q3.Count Occurrence
        You are given an array A containing N integer elements and an integer K, and your task is to return the count of occurrences of K in array A.
        Note: You have to complete findCount function. No need to take any input.
    

    
        const findCount = (N, K, Arr) => 
        {
        let count=0;
        for (let i=0;i< N ;i++){
        if(K===Arr[i]){
        count=count+1;
        }
    
        }
        return count;
        };
    

    
Q4.Even Odd
        You are given an array A containing N integer elements, and your task is to return an array B(having a size equal to), where B[0]contains the sum of all even elements of array A and B[1]Mhas the sum of all odd elements of the array A.
        Note: You have to complete findEvenOdd function. No need to take any input.
    

    
        const findEvenOdd = (N, Arr) => 
        {
        let B=[]
        var a=Arr.filter((x)=>x%2===0) 
        var b=a.reduce((acc,cum)=>acc=acc+cum,0)
        B.push(b)
    
        var c=Arr.filter((x)=>x%2!==0)
        var d=c.reduce((acc,cum)=> acc=acc+cum,0)
        B.push(d)
        return B;
        };
